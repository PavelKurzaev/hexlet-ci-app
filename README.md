# Example app for CI Hexlet course

Starting boilerplate of [Strapi](https://strapi.io/) application

## System requirements

* NodeJS >= 12 <= 16
* NPM >= 6.x
* Make

## Using

```sh
make setup
make start
```

## Run tests

```sh
make test
```

## Run linter

```sh
make lint
```
